## 📌 Project Description

This project showcases a professional manual testing suite conducted on the SHEIN web application. It involves a systematic approach to quality assurance, covering functional, UI/UX, and regression testing scenarios. The primary goal was to ensure the stability of the user journey from landing to checkout.

---

## 🛠 Testing Modules & Scope

- **Navigation & Home Page:** Verification of informational banners, redirection links, and logo functionality.
- **Search Engine:** Testing keyword suggestions, dynamic placeholders, and "No Results Found" logic.
- **Account Management:** Validation of Sign-In, Registration, and "Forgot Password" workflows.
- **Shopping Cart:** Verification of product quantities, subtotal calculations, and item removal.
- **Wishlist (Favorites):** Testing real-time updates and interaction between the cart and favorites.

---

## 🧪 Methodologies & Techniques Applied

- **Black Box Testing:** Functional testing performed via the UI.
- **Positive & Negative Testing:** Validated both expected behavior and error handling for invalid inputs.
- **Boundary Value Analysis (BVA):** Tested quantity limits (e.g., Qty 99) and stock availability messages.
- **Error Guessing:** Specifically targeted form validation and system feedback delays.
- **UI/UX Verification:** Evaluated modal behaviors and dynamic UI updates (Side Cart).
